foley.base

Canonical data models for foley — the single source of truth (SSOT) types.

This module is stdlib-only and declarative: it defines the dataclasses, enums, and affordance registries every other layer shares, plus generic dict/JSON (de)serialization. It contains NO policy (see foley.licensing) and NO I/O (see foley.stores / foley.audio).

Serialization contract:

  • record.to_dict() -> a plain dict (nested dataclasses recursed via dataclasses.asdict; enum members preserved, and JSON-safe because every enum subclasses str).

  • record.to_json() -> a JSON string.

  • Cls.from_dict(d) / Cls.from_json(s) -> reconstructs, coercing enum fields from their string values and nested LicenseRecord from its dict. Unknown keys are ignored (forward-compatible schema evolution); missing keys fall back to field defaults.
