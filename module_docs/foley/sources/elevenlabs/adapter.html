foley.sources.elevenlabs.adapter

ElevenLabs Sound Effects generate adapter — hosted Text-to-SFX v2 over HTTP.

Turns a natural-language prompt into a GeneratedClip via the ElevenLabs POST /v1/sound-generation endpoint (report 02 §ElevenLabs). The response is raw audio bytes (default MP3 44.1 kHz), stored by-value (the generation flywheel) once the foley.sources.generate.generate() façade routes it through ingest_one().

Two ElevenLabs-specific quirks the adapter handles:

  • ``output_format`` is a URL query parameter, not a JSON body field; everything else (text / model_id / prompt_influence / loop / duration_seconds) is the JSON body.

  • No seed / negative-prompt / step control — generation is non-deterministic, so generation_seed stays None and those unified affordances warn-and-drop.

HTTP is dependency-injected (a Transport), so the adapter is fully testable with no network and import foley stays dol-only; the real requests lives only behind requests_transport() (the foley[elevenlabs] extra). The adapter performs NO storage/library access — it builds the audio + a generated LicenseRecord and returns; the façade converges it on the shared ingest pipeline.

generate(prompt: str, *, duration: float | None = None, prompt_influence: float = 0.3, negative_prompt: str | None = None, steps: int | None = None, seed: int | None = None, loop: bool = False, output_format: str = 'wav', **kw) → GeneratedClip[source]

Generate a sound effect for prompt; return its bytes + provisional candidate.

Parameters:

  • prompt – The natural-language sound description (required).

  • duration – Seconds (clamped to the native 0.5..30 range). None lets the model auto-determine the length from the prompt.

  • prompt_influence – 0..1 (identity map to the native field; higher = closer to the prompt, less variety). Default 0.3.

  • seed (negative_prompt / steps /) – No ElevenLabs SFX equivalent — a non-default value warns-and-drops (recorded in the clip notes); generation_seed stays None.

  • loop – Produce a seamless-loopable clip (native loop; v2 default).

  • output_format – Coarse foley token (wav | opus | mp3) translated to the native enum; wav falls back to MP3 (the SFX endpoint has no lossless container) and is re-archived to FLAC by ingest.

  • **kw – Extra/unknown affordances (warn-and-dropped).

Returns:

A GeneratedClip (origin=generated, license_id='ElevenLabs-SFX', is_ai_generated=True).
