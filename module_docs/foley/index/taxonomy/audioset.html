foley.index.taxonomy.audioset

AudioSet-label -> UCS-CatID overlap map (report 04 §5.3, [11 §3.1]).

The machine-label layer (PANNs/PaSST emit AudioSet labels on ingest) is bridged to the human-facing UCS browse tree by a small overlap map. This is the conceptual seed of the EnvSound-UCS “Rosetta stone”; the full table drops in as taxonomy/data/audioset_ucs.json and merges over the seed. Every target CatID is validated against the UCS table at load time (fail-fast on a broken map).

Build the AudioSet(name|MID) -> UCS-CatID map.

Parameters:

  • data_dir – Directory to look for audioset_ucs.json in (defaults to the package’s taxonomy/data/); merged over the seed when present.

  • table – The UCS table every target CatID must exist in (defaults to default_ucs_table()).

  • include_seed – Start from the in-code seed map (default True).

Returns:

A ready AudioSetUcsMap.

Raises:

ValueError – If any entry targets a CatID absent from table.
